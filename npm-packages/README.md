This contains supporting NPM Packages and scripts to generate them.
All packages defined here are scoped in the [`elite-dangerous-plugin-framework`](https://www.npmjs.com/org/elite-dangerous-plugin-framework) npm-Organization.

# Packages

## `@elite-dangerous-plugin-framework/core`

The main package. Contains an abstract class definitions extending `HTMLElement` you can extend for your Main Plugin and Settings Classes. Doing so will give you correct type annotations for properties / methods that EDPF cares about. 


It will also give you Type annotations for the Journal (via `@elite-dangerous-plugin-framework/journal`)

## `@elite-dangerous-plugin-framework/journal`
This is an autogenerated module, which uses [Jixxed's collection of Journal Schemas](https://jixxed.github.io/ed-journal-schemas/) as a base to generate TypeScript Types.

This is also why its referenced as a Git Submodule here.

Besides thje Types, this also exposes functions to convert the stringified JSON into a regular JSON or a BigInt-compatible JSON

# Versioning

Each Module has it's own approach to versioning.

`@elite-dangerous-plugin-framework/core` follows SemVer,
`@elite-dangerous-plugin-framework/journal` uses a variant of CalVer: `0.YYYYMMDD`
